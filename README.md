# Personal Portfolio

Starter project untuk website portofolio pribadi.

## Tech Stack

- Vite
- React 19
- TypeScript
- Tailwind CSS v4
- Bun

## Prerequisites

- Bun sudah terpasang (`bun --version`)

## Menjalankan Project

```bash
bun install
bun dev
```

App akan berjalan di URL local yang ditampilkan Vite (umumnya `http://localhost:5173`).

## Available Scripts

- `bun dev` - Menjalankan development server.
- `bun run build` - Type-check dan build production ke folder `dist`.
- `bun run preview` - Menjalankan preview hasil build.
- `bun run lint` - Menjalankan ESLint.

## Struktur Singkat

- `src/App.tsx` - Layout dan konten utama portofolio.
- `src/index.css` - Import Tailwind CSS.
- `vite.config.ts` - Konfigurasi Vite + plugin Tailwind.

## Catatan

- Silakan ubah konten di `src/App.tsx` sesuai profil, project, dan kontak kamu.
